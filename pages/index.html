脑控大师 - 自定义预设指南

从 v2.1.2 版本开始，脑控大师支持用户自定义预设模式和提示词模板。

如何添加自定义预设

  1. 在 AstrBot WebUI 中进入「插件管理」
  2. 找到「脑控大师」插件，点击配置
  3. 找到「自定义预设模式」配置项，点击添加
  4. 填写以下字段：

配置字段说明

字段 说明 示例
name 预设名称（用于在「预设模式」中选择） maid
enter 进入沉浸状态时的提示词模板 你现在是女仆...
afterglow 退出后的余韵阶段提示词 你还在女仆状态...
exit 完全退出后的提示词 你恢复正常了

提示词模板占位符

  • {item_name} - 装置名称（来自配置项「装置名称」）
  • {sensitivity} - 当前敏感度（0-100）

示例：女仆模式

{
  "name": "maid",
  "enter": "你现在是可爱的小女仆，当前敏感度 {sensitivity}/100。请用「主人」「请」等女仆语气说话，敏感度越高越容易害羞和颤抖。",
  "afterglow": "你还在女仆状态中，偶尔会下意识用女仆语气说话。",
  "exit": "你恢复了正常身份。"
}
提示：如果自定义预设的提示词留空，插件会自动 fallback 到 control 模式，避免注入空提示词。

使用自定义预设

添加完成后，在「预设模式」配置项中直接输入你自定义的名称（例如 maid），即可切换到该预设。
